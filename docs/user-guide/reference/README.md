## Čo je referenčný dokument?

Referenčný dokument je **mapou**, nie cestou. Nevedie ťa krok za krokom — predpokladá, že vieš, čo hľadáš, a chce ti to dať rýchlo a presne.

Referencia je typicky:

- Tabuľka so zoznamom hodnôt + ich významov
- Diagram s prechodmi stavov
- Slovník (abecedne usporiadané pojmy)
- Reference pre API/SDK
