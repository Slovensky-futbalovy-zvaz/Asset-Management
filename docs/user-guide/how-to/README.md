## Čo je how-to návod?

How-to je **recept**. Začína sa predpokladmi („čo musíš mať predtým, ako začneš"), pokračuje očíslovanými krokmi, a končí sekciou „Možné problémy".

How-to **nie je**:

- ❌ Tutoriál pre začiatočníkov → tie sú v [Getting Started](../getting-started/)
- ❌ Vysvetlenie kontextu → to je v [Use Cases](../use-cases/)
- ❌ Slovník či referenčná tabuľka → to je v [Reference](../reference/)
