BEZPEČNOSŤ

Bezpečnosť je first-class concern.

Nie je to feature, ktorú riešime na konci. Je to vstavané v architektúre.

🛡️ RBAC na úrovni routes

Tri roly: EMPLOYEE, ASSET_MANAGER, ADMIN. Každá HTTP route má explicitne deklarovanú požadovanú rolu. Fastify middleware odmietne neautorizovaný request skôr než sa vôbec dostane k business logike.

📝 Append-only audit log

Každá CRUD operácia sa zapíše do audit_log kolekcie spolu s actor-om, cieľom, timestampom, IP adresou a JSON diff-om zmien. Logy sa nedajú meniť ani mazať cez verejné API — len cez admin nástroje s plnou auditovateľnosťou.

🔒 End-to-end TLS

HTTPS všade. Vercel automaticky generuje TLS certifikáty cez Let's Encrypt. MongoDB Atlas používa TLS pre database connections. JWT tokens sú podpísané asymetrickou kryptografiou.

🗄️ Encrypted at rest

MongoDB Atlas šifruje všetky dáta na disku cez AES-256. Vercel filesystem storage tiež šifrovaný. Žiadne nešifrované backupy alebo logy.

⚙️ MongoDB transactions

Kritické operácie (vytvorenie majetku + audit log, schválenie výpožičky + statusy) bežia v transakciách. Žiadne čiastočne aplikované zmeny — buď všetko, alebo nič.

🎯 Coordinated Vulnerability Disclosure

Verejne dokumentovaný CVD policy v SECURITY.md. Bezpečnostné správy môžu byť posielané cez GitHub Security Advisories alebo na email.
