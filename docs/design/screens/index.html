Technický prehľad

Čo je v týchto mockup-och technicky a strategicky.

✓ Multi-tenant white-label

  • Brand identita cez CSS custom properties (4 tenanti × 4 farby)
  • Tenant switching v real-time bez page reload-u
  • Per-tenant demo data — každý tenant vidí svoje reálne assets
  • Logická multi-tenancy podľa ADR-0010

✓ Mobile-first responsive

  • 3 viewporty: 375 (mobile) / 768 (tablet) / 1280 (desktop)
  • Bottom navigation na mobile, sidebar na desktop
  • Bottom sheet filtre, swipe-friendly cards
  • Touch targets ≥ 44 × 44 px (WCAG)

✓ Role-aware UI (RBAC)

  • Employee · Asset Manager · Admin — 3 roly v jednom codebase
  • Conditional rendering podľa data-role attribute-u
  • Sidebar menu sa adaptuje podľa role
  • Audit log tab visible len pre Asset Manager+

✓ Real interakcie

  • Live search filter (Assets list)
  • Multi-step wizard s progress trackerom (Loan request)
  • 5-tab navigation s URL state (Asset detail)
  • Real SVG QR code generation (qrcode-generator lib)
